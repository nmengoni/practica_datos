# Práctica de Datos

Repositorio orientado a la práctica de conceptos y ejercicios de ciencia de datos.

## Descripción

Este proyecto reúne materiales y ejemplos para explorar tareas frecuentes de análisis de datos, con foco en aprendizaje práctico y experimentación.

## Objetivos

- Practicar flujo de trabajo en proyectos de datos.
- Explorar técnicas básicas de limpieza, análisis y visualización.
- Documentar avances de forma ordenada y reproducible.

## Requisitos

- Python 3.10 o superior.
- Entorno virtual recomendado (`venv` o similar).
- Dependencias del proyecto (cuando aplique).

## Uso

1. Clonar este repositorio.
2. Crear y activar un entorno virtual.
3. Instalar dependencias.
4. Ejecutar notebooks o scripts según la práctica.
